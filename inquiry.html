Inquiry and Analysis

Explanation and Justification

Explain: The World Heath Organization website currently contains massive amounts of data about a multitude of health related topics. There are many important topics that can range from diseases to life expectancy. All this data is organized by country and can be easily accessed through the World Health Organization data repository. Specifically, the statistics about homicides are located in section 16.1 Violence which will be the main focus for the later part of this project. The thousands of data cells about homicides by country are all compacted into a single JSON file. A JSON file is a file that stores simple data structures and objects in JavaScript Object Notation (JSON) format, which is a standard data interchange format. It is primarily used for transmitting data between a web application and a server.

Justify: Much of the tabular data presented in the JSON files found on the World Heath Organization website are visually unappealing and difficult to understand because of the lack of presentation. Due to this flaw, many spectators are unable to learn from the health data as desired. However, due to the power of web development, this valuable data can be transformed into an athaestically pleasing website that is comprehensible. HTML can be used to build the structure of the website, CSS can be responsible for the stylistic choices, and JavaScript can be utilized for behavioural aspects of the website. Mainly, the HTML canvas tag opens up opportunities for a variety of options to view data graphically. It is extremely important to be able to provide a graphical solution to JSON data so that the target audience is able to understand and learn information from the World Heatlh Organization.

Target Audience: The target audience for this solution is mainly teen students. The grade range for this target audience is from Grade 8 - Grade 10.

Population

Population.io is an outstanding website that integrates a variety of interesting features from different aspects of web development. As soon as the user opens the website, an animation and counter of our world population is displayed on the screen recording the global birth rate in real time. Everytime a baby is born an icon of a baby pops up and their ethnicity is listed as well as “Hello” in different languages according to the baby’s ethnicity. Also, beneath the animation is an input bar where the user can type in personal information to predict their life expectancy and look at their life’s journey. It is easy to navigate through many of the diverse functionality and data. CSS is used repeatedly to create transitions, colours, and a great view for the user. Overall population.io is a great website that utilizes HTML, CSS, and JavaScript to the fullest.
